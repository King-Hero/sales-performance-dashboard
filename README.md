# 📊 Sales Performance Dashboard

## 🔷 Objective
This project focuses on analyzing sales data to generate actionable business insights through an interactive dashboard.

## 🔷 Tools Used
- Power BI
- Data Cleaning & Transformation
- DAX (Data Analysis Expressions)

## 🔷 Key Features
- Revenue, Profit, and Quantity KPIs
- Monthly, Quarterly, and Yearly Sales Trends
- Top-performing and Low-performing Products
- Region-wise and Category-wise Analysis
- Interactive Filters for dynamic exploration

## 🔷 Key Insights
- A small number of products contribute to the majority of revenue
- Certain regions consistently outperform others
- Sales trends show clear seasonal patterns
- Some categories underperform and require strategic focus
